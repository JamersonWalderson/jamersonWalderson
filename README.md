# 💫 About Me:
Sou formado em Sistemas de Informação, pela Estácio.<br><br>Atualmente estou empregado como Analista de Sistemas e também atuo como freelancer nas horas vagas.<br><br>Iniciei minha carreira no Ministério Público do Trabalho em Alagoas, como Estagiário de TI, onde trabalhei com Suporte ao usuário e desenvolvimento de soluções em JavaScript e jQuery para sistemas internos.<br><br>Tenho experiência em:<br>Front-end (Bootstrap, Next.js, Alpine.js (semelhante a Vue.js), Reactjs, JQuery e tailwind)<br>Mobile: React native (nativo e expo)<br>Backend: Python (Django, Flask), PHP (Laravel), Node, Pocketbase(BaaS), Firebase(BaaS)<br>Testes: Jest, Pytest, Cypress, React native testing library, Debugging (Chrome, VS Code)<br>Banco de dados: MongoDB(RealmDB), PostgreSQL, Pocketbase, Firebase<br>Versionamento: Git.<br><br>*Além disso, tenho conhecimentos em DevOps que me permitem configurar ambientes com Docker, Nginx, e Portainer embora não seja meu foco ou minha área de especialização<br>DevOps: Docker, Nginx (nativo ou proxy manager), Portainer, firewall (UFW), Cloudflare(para proxy reverso), Hostinger, registrobr, Google Play Console, N8N<br>
